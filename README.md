**Q: IPMI 是什么？为什么重要？**  
A: IPMI 是 Intelligent Platform Management Interface 的缩写，可以让你在操作系统之外远程管理服务器。即使服务器死机或系统崩溃，你也能通过 IPMI 重启、查看控制台、重装系统。对于自管理服务器来说，这是必备功能。

**Q: 可以安装 Windows 系统吗？**  
A: VPS 支持 Windows。独立服务器理论上也可以装 Windows，但需要你自己有许可证，并且可能需要通过 IPMI 手动安装。

**Q: 带宽和流量是怎么计算的？**  
A: 默认配置下，VPS 和独立服务器都有带宽限制（150Mbps）和流量限制（1TB到10TB不等）。如果流量用完了，速度会被限制但不会额外收费。你可以付费升级到 1Gbps 端口或 100Mbps 不限流量。
